기본 용어 정리
   1. MFC(Microsoft Foundation Classes) : window 에서 제공하는 c++기반 ui 라이브 러리 
   2. 메세지 핸들러 : 메세지를 받고 동작을 결정함
   3. 윈도우 프로시져 : 메세지 핸들러의 집합
   4. DLL(Dynamic-Link Library) : 프로그램 실행중 결합하여 사용할 수 있는 코드와 리소스의 집합
   5. 장치 독립성 : 윈도우 응용프로그램은 비디오카드,프린터등 주변장치가 바꿔도 드라이버 설치
                 이후 프로그램 변경 없이 사용 가능. 응용 프로그램은 주변 장치를 api통해
		 통신 함으로 장치 독립성을 지님
   6. RAD(Rapid Application Development): 화면을 디자인하고 코드를 추가해 개발하는 방식
   7. .NET : 윈도우 OS에 설치할 수 있는 소프트웨어 개발 및 실행 환경, 런타임이라는 가상 머신을 제어하여
             응용 프로그램이 구동됨
